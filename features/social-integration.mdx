## Complete Implementation

### HTML Structure

The social media section is organized as a dedicated section with flexbox layout:

```html Actividad_Css.html:213-227
<section id="box-model">
    <h2>Redes Sociales</h2>
    <div class="box">
        <h2>Sígueme</h2>
        <a href="https://www.facebook.com/profile.php?id=100009198955739&locale=es_LA" target="_blank">
            <img src="img/facebook.png.png" alt="Facebook" width="30">
        </a>
        <a href="https://www.instagram.com/fer.colombia/?next=%2F" target="_blank">
            <img src="img/instagram.png.png" alt="Instagram" width="30">
        </a>
        <a href="https://www.linkedin.com/in/luisa-fernanda-zuleta-hurtado-4148b22b5/" target="_blank">
            <img src="img/linkedin.png.png" alt="Linkedin" width="30">
        </a>
    </div>
</section>
```

<Note>
**Key Features**:
- All links use `target="_blank"` to open in new tabs
- Semantic HTML with proper `alt` attributes for accessibility
- Consistent icon size (30px width)
- Grouped within a `.box` container for unified styling
</Note>

### CSS Styling

#### Container Box

The social icons are wrapped in a flexbox container with gradient background:

```css Actividad_Css.html:74-86
.box {
    width: 300px;
    padding: 20px;
    margin: 30px auto;
    border: 4px solid #fbfdfb;
    border-radius: 10px;
    background-color: #c5eaf8;
    box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.2);
    display: flex;  /* Activa Flexbox */
    gap: 15px;  /* Espacio entre íconos */
    justify-content: center; /* Alinea los íconos en el centro */
    align-items: center; /* Centra los íconos verticalmente */
}
```

<Tip>
**Design Elements**:
- `display: flex` with `gap: 15px` creates consistent spacing
- `justify-content: center` horizontally centers all icons
- `box-shadow` adds depth and elevation
- `border-radius: 10px` softens the corners
</Tip>

#### Link Styling

```css Actividad_Css.html:88-98
.box a {
    font-size: 30px;
    color: #333; /* Color de los íconos */
    text-decoration: none;
    transition: color 0.3s ease-in-out, transform 0.3s ease-in-out;
}

.box a:hover {
    color: #007bff; /* Cambia de color al pasar el mouse */
    transform: scale(1.2); /* Hace un efecto de agrandamiento */
}
```

<Card title="Interactive Hover Effect" icon="hand-pointer">
When users hover over icons:
- Color changes from `#333` (dark gray) to `#007bff` (blue)
- Icon scales up by 20% using `transform: scale(1.2)`
- Smooth 0.3s transition for both properties
</Card>

#### Image Hover Effect

```css Actividad_Css.html:101-109
.box img {
    width: auto; 
    height: 40px; 
    transition: transform 0.3s ease-in-out;
}

.box img:hover {
    transform: scale(1.2); 
}
```

## Best Practices

### Link Management

<Note>
**Security Consideration**: When using `target="_blank"`, consider adding `rel="noopener noreferrer"` to prevent security vulnerabilities:

```html
<a href="..." target="_blank" rel="noopener noreferrer">
```

This prevents the new page from accessing the `window.opener` object.
</Note>

## Mobile Responsiveness

The social box maintains its centered layout on all screen sizes:

```css
.box {
    width: 300px;
    margin: 30px auto; /* Centers the box */
}
```

On screens below 600px, the flexbox container adapts:

```css Actividad_Css.html:139-146
@media (max-width: 600px) {
    .flex-container {
        flex-direction: column;
    }
}
```

<Note>
While the general flex containers switch to column layout on mobile, the `.box` social icons maintain their horizontal row layout for better usability.
</Note>
