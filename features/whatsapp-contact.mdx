## WhatsApp URL Format

The link uses WhatsApp's short URL service:

```
https://w.app/virtualmind
```

<Note>
**Alternative URL Formats**:

WhatsApp supports multiple URL patterns:

1. **wa.me format** (with phone number):
   ```
   https://wa.me/573001234567
   ```

2. **wa.me with message** (pre-filled text):
   ```
   https://wa.me/573001234567?text=Hola%2C%20me%20interesa%20tu%20portafolio
   ```

3. **w.app custom link** (branded shortlink):
   ```
   https://w.app/virtualmind
   ```

The portfolio uses option 3, which provides a clean, branded experience.
</Note>

## Mobile Optimization

### Responsive Considerations

```css
.whatsapp-box {
    padding: 10px 450px; /* Desktop */
}
```

<Note>
**Potential Issue**: The horizontal padding of `450px` may be too large for mobile screens. Consider adding a media query:

```css
@media (max-width: 600px) {
    .whatsapp-box {
        padding: 10px 20px;
    }
}
```

This would ensure the button remains usable on smaller screens.
</Note>

### Mobile Behavior

On mobile devices:
- The `w.app` link automatically detects the device
- Opens the WhatsApp app directly (if installed)
- Falls back to WhatsApp Web if app not available
- Maintains chat history and conversation context

## Security Considerations

<Note>
**Recommendation**: Add `rel="noopener noreferrer"` to the WhatsApp link:

```html
<a href="https://w.app/virtualmind" target="_blank" rel="noopener noreferrer">Contáctame</a>
```

This prevents the opened page from accessing the portfolio's `window.opener` object, protecting against potential security vulnerabilities.
</Note>
