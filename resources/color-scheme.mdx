## Overview

The portfolio uses a cohesive color scheme built around cyan, blue, and green tones, creating a modern and professional aesthetic. The design leverages gradients extensively to add depth and visual interest.

### Gradient Backgrounds

<Note>
The portfolio makes extensive use of linear gradients to create dynamic, modern backgrounds.
</Note>

#### Main Portfolio Gradients

**Body Background - Main Portfolio**
```css
background: linear-gradient(to left, rgba(5, 240, 248, 0.849), rgba(176, 208, 212, 0.7));
```
- **From:** `rgba(5, 240, 248, 0.849)` - Bright cyan with transparency
- **To:** `rgba(176, 208, 212, 0.7)` - Muted blue-gray with transparency
- **Direction:** Right to left

**Header Gradient**
```css
background: linear-gradient(90deg, #5eabf3, #c4f1ef);
```
- **From:** `#5eabf3` - Blue
- **To:** `#c4f1ef` - Light cyan
- **Direction:** Left to right (90deg)

**Section Background**
```css
background: linear-gradient(to right, #02b5d4, #dad8d7);
```
- **From:** `#02b5d4` - Dark cyan
- **To:** `#dad8d7` - Light gray
- **Direction:** Left to right

**Column Gradient**
```css
background: linear-gradient(to right, #f8f6f6, #1cc7f1);
```
- **From:** `#f8f6f6` - Off-white
- **To:** `#1cc7f1` - Bright cyan
- **Direction:** Left to right

**WhatsApp Box Gradient**
```css
background: linear-gradient(to right, #0a0a0a, #52dff5);
```
- **From:** `#0a0a0a` - Nearly black
- **To:** `#52dff5` - Cyan
- **Direction:** Left to right

## Shadow Effects

The portfolio uses subtle shadows to add depth:

```css
box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.2);
```

- **Offset:** 5px horizontal, 5px vertical
- **Blur:** 15px
- **Color:** Black with 20% opacity

## Usage Guidelines

<Note>
All gradients use CSS linear-gradient with directional control (to left, to right, 90deg) to maintain consistency across the design.
</Note>

### Color Accessibility

- Primary text colors (`#141018`, `#333`) provide strong contrast against light backgrounds
- White text is used exclusively on dark gradient backgrounds for readability
- Link colors (`#335275`) are distinguishable from regular text

### Implementation Example

```html
<!-- Header with gradient -->
<header style="background: linear-gradient(90deg, #5eabf3, #c4f1ef);">
  <h1 style="color: rgb(36, 55, 73);">Luisa Fernanda Zuleta Hurtado</h1>
</header>

<!-- Section with gradient background -->
<section style="background: linear-gradient(to right, #02b5d4, #dad8d7); color: rgb(24, 22, 22);">
  <h2>Perfil Profesional</h2>
  <p>Content here...</p>
</section>
```
