## Image Styling

### Global Image Styles

```css
img {
    max-width: 100%;
    height: auto;
    display: block;
    margin: 0 auto;
    border-radius: 5px;
}
```

**Properties:**
- **max-width:** 100% - Responsive, never exceeds container width
- **height:** auto - Maintains aspect ratio
- **display:** block - Prevents inline spacing issues
- **margin:** 0 auto - Centers images horizontally
- **border-radius:** 5px - Subtle rounded corners

### Social Media Icon Styles

#### Box Container Icons

```css
.box img {
    width: auto; 
    height: 40px; 
    transition: transform 0.3s ease-in-out;
}

.box img:hover {
    transform: scale(1.2); 
}
```

**Features:**
- Height fixed at 40px, width scales automatically
- Smooth hover animation (0.3s)
- 20% scale increase on hover for interactivity

#### WhatsApp Icon Special Styling

```css
.whatsapp-box img {
    width: 70px;
    height: auto;
}
```

**Features:**
- Larger size (70px) to emphasize call-to-action
- No hover effect (applied to parent container instead)
