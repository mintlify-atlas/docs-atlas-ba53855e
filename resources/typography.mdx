## Overview

The portfolio employs a carefully selected typography system that balances professionalism with readability. Different font families are used strategically across various HTML files to create distinct visual identities for different sections.

## Font Families

### Main Portfolio (Actividad_Css.html)

<Card title="Primary Font Stack" icon="text-size">
```css
font-family: Georgia, 'Times New Roman', Times, serif, sans-serif;
```
**Applied to:** Body text  
**Fallback chain:** Georgia → Times New Roman → Times → serif → sans-serif  
**Character:** Classic, professional, elegant
</Card>

<Note>
This serif font choice gives the portfolio a sophisticated, professional appearance while maintaining excellent readability.
</Note>

## Text Styling & Effects

### Text Decoration

**Navigation Links**
```css
nav ul li a {
    text-decoration: none;
}
```
- Clean, modern link appearance without underlines
- Applied to all navigation elements

**Social Media Links**
```css
.box a {
    text-decoration: none;
}
```
- Icon links without decoration for cleaner look

### Text Alignment

**Centered Text**
```css
body {
    text-align: center;
}
```
- All portfolio pages use centered text alignment
- Creates a focused, symmetrical layout

**Footer Alignment (index.html)**
```css
#footer {
    text-align: center;
}
```

## Hover Effects

### Link Color Transitions

```css
.box a {
    transition: color 0.3s ease-in-out, transform 0.3s ease-in-out;
}

.box a:hover {
    color: #007bff;
    transform: scale(1.2);
}
```

- Smooth color transition over 0.3 seconds
- Hover color: Bootstrap blue (`#007bff`)
- Combined with scale transformation for emphasis

## Responsive Typography

<Note>
While explicit responsive font sizes aren't defined, the use of `em` units in headers allows for proportional scaling based on parent elements.
</Note>

```css
@media (max-width: 600px) {
    .flex-container {
        flex-direction: column;
    }
}
```

Mobile devices inherit the same font sizes but benefit from layout adjustments.
