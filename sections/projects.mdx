## Styling

The section uses two main CSS classes:

### Container Grid

```css
.contenedor {
    display: grid;
    grid-template-columns: 1fr 1fr; /* Divide en dos columnas iguales */
    gap: 10px; /* Espaciado entre columnas */
    max-width: 400px;
    margin: auto;
}
```

<Accordion title="Grid Properties Breakdown">
- **display: grid** - Enables CSS Grid layout
- **grid-template-columns: 1fr 1fr** - Creates two equal columns
- **gap: 10px** - 10px spacing between columns
- **max-width: 400px** - Limits container width
- **margin: auto** - Centers the container
</Accordion>

### Column Styling

```css
.columna {
    background: linear-gradient(to right, #f8f6f6, #1cc7f1);
    padding: 10px;
    border-radius: 8px;
}
```

<Note>
The `.columna` class applies a light gray to cyan gradient background with rounded corners, creating visually distinct content blocks.
</Note>
