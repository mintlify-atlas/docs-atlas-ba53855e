## Styling

### Social Media Box (.box)

```css
.box {
    width: 300px;
    padding: 20px;
    margin: 30px auto;
    border: 4px solid #fbfdfb;
    border-radius: 10px;
    background-color: #c5eaf8;
    box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.2);
    display: flex;  /* Activa Flexbox */
    gap: 15px;  /* Espacio entre íconos */
    justify-content: center; /* Alinea los íconos en el centro */
    align-items: center; /* Centra los íconos verticalmente */
}
```

<Accordion title="Box Styling Breakdown">
- **width: 300px** - Fixed width for consistent sizing
- **padding: 20px** - Internal spacing
- **margin: 30px auto** - Centered with vertical spacing
- **border: 4px solid #fbfdfb** - Light border for definition
- **border-radius: 10px** - Rounded corners
- **background-color: #c5eaf8** - Light blue background
- **box-shadow** - Subtle shadow for depth
- **display: flex** - Flexbox for icon alignment
- **gap: 15px** - Spacing between icons
- **justify-content: center** - Horizontal centering
- **align-items: center** - Vertical centering
</Accordion>

### Link Hover Effects

```css
.box a {
    font-size: 30px;
    color: #333; /* Color de los íconos */
    text-decoration: none;
    transition: color 0.3s ease-in-out, transform 0.3s ease-in-out;
}

.box a:hover {
    color: #007bff; /* Cambia de color al pasar el mouse */
    transform: scale(1.2); /* Hace un efecto de agrandamiento */
}
```

<Note>
Hover effects include color change to blue (#007bff) and 20% scale increase, providing visual feedback for interaction.
</Note>
