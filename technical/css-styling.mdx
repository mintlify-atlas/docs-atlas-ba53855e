## Overview

The portfolio employs embedded CSS styling with modern techniques including gradients, flexbox, grid, animations, and responsive design. All styles are defined within `<style>` tags in the document head.

## CSS Architecture

### Embedded Styles

All three portfolio versions use embedded stylesheets within the HTML document:

```html
<head>
    <style>
        /* All CSS rules defined here */
    </style>
</head>
```

<Note>
While external stylesheets are generally preferred for production, embedded styles are used here for simplicity and portability of the single-page portfolio.
</Note>

## Selectors and Specificity

### Element Selectors

Basic styling applied to fundamental HTML elements:

```css
body {
    font-family: Georgia, 'Times New Roman', Times, serif, sans-serif;
    margin: 40px;
    padding: 40px;
    background: linear-gradient(to left, rgba(5, 240, 248, 0.849), rgba(176, 208, 212, 0.7));
    color: #333;
    text-align: center;
}

h1 {
    color: rgb(20, 16, 24);
    font-size: 40px;
}

img {
    max-width: 100%;
    height: auto;
    display: block;
    margin: 0 auto;
    border-radius: 5px;
}
```

**Key Techniques:**
- **Font stacking** with fallback fonts
- **Responsive images** using `max-width: 100%`
- **Auto margins** for horizontal centering
- **Border radius** for rounded corners

### RGBA Color Values

Transparency effects using RGBA notation:

```css
/* Background with transparency */
background-color: rgba(185, 234, 253, 0.7);

/* Gradient with alpha channel */
background-image: linear-gradient(to right, rgba(44, 54, 58, 0.7), rgba(185, 116, 253, 0.7));

/* Box shadow with transparency */
box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.2);
```

<Note>
RGBA values allow colors with an alpha channel (transparency). The fourth parameter ranges from 0 (fully transparent) to 1 (fully opaque).
</Note>

### RGB Color Values

## Box Model Implementation

### Standard Box Model

The portfolio demonstrates all aspects of the CSS box model:

```css
.box {
    width: 300px;              /* Content width */
    padding: 20px;             /* Inner spacing */
    margin: 30px auto;         /* Outer spacing + horizontal centering */
    border: 4px solid #fbfdfb; /* Border width, style, and color */
    border-radius: 10px;       /* Rounded corners */
}
```

**Box Model Breakdown:**
- **Content**: 300px wide
- **Padding**: 20px on all sides
- **Border**: 4px solid border
- **Margin**: 30px top/bottom, auto left/right (centers element)

### Box Shadow

3D depth effect using box-shadow:

```css
.box {
    box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.2);
}
```

**Parameters:**
1. **5px** - Horizontal offset (right)
2. **5px** - Vertical offset (down)
3. **15px** - Blur radius
4. **rgba(0, 0, 0, 0.2)** - Shadow color with 20% opacity

### Overflow Control

```css
.box {
    overflow: hidden;
}
```

Prevents content from spilling outside the border-radius boundary.

## Layout Systems

### Flexbox Implementation

The portfolio uses Flexbox for flexible, one-dimensional layouts:

#### Social Media Icons

```css
.box {
    display: flex;
    gap: 15px;
    justify-content: center;
    align-items: center;
}
```

**Flexbox Properties:**
- `display: flex` - Activates flexbox layout
- `gap: 15px` - Spacing between flex items
- `justify-content: center` - Horizontal centering
- `align-items: center` - Vertical centering

#### Flex Container with Multiple Items

```css
.flex-container {
    display: flex;
    justify-content: space-around;
    align-items: center;
    padding: 20px;
}

.flex-item {
    width: 100px;
    height: 100px;
    background-color: #4CAF50;
    color: white;
    display: flex;
    justify-content: center;
    align-items: center;
    font-size: 1.5em;
}
```

<Note>
`justify-content: space-around` distributes items evenly with equal space around each item.
</Note>

#### WhatsApp Contact Box

```css
.whatsapp-box {
    display: flex;
    align-items: center;
    justify-content: center;
    gap: 0px;
    background: linear-gradient(to right, #0a0a0a, #52dff5);
    padding: 10px 450px;
    border-radius: 10px;
}
```

### Grid Implementation

CSS Grid for two-dimensional layouts:

#### Two-Column Layout

```css
.contenedor {
    display: grid;
    grid-template-columns: 1fr 1fr;  /* Two equal columns */
    gap: 10px;                        /* Space between columns */
    max-width: 400px;
    margin: auto;
}

.columna {
    background: linear-gradient(to right, #f8f6f6, #1cc7f1);
    padding: 10px;
    border-radius: 8px;
}
```

**Grid Properties:**
- `grid-template-columns: 1fr 1fr` - Creates two equal-width columns using fractional units
- `gap: 10px` - Gutter spacing between grid items

#### Three-Row Layout

```css
.grid-container {
    display: grid;
    grid-template-rows: auto 1fr auto;  /* Header, expandable main, footer */
    gap: 10px;
    padding: 20px;
}

.grid-container header,
.grid-container main,
.grid-container footer {
    padding: 20px;
    background-color: #2E8B57;
    color: white;
}
```

<Note>
The `1fr` unit in the middle row makes the main content area expand to fill available space.
</Note>

## Interactive Elements

### Hover Effects

Transforms and color changes on hover:

```css
.box a:hover {
    color: #007bff;
    transform: scale(1.2);
}

.box img:hover {
    transform: scale(1.2);
}

.whatsapp-box:hover {
    transform: scale(1.1);
}
```

**Transform Effects:**
- `scale(1.2)` - Enlarges element to 120% of original size
- `scale(1.1)` - Enlarges to 110% for subtle effect

### Transitions

Smooth animations for interactive elements:

```css
.box a {
    transition: color 0.3s ease-in-out, transform 0.3s ease-in-out;
}

.box img {
    transition: transform 0.3s ease-in-out;
}

.whatsapp-box {
    transition: transform 0.3s ease-in-out;
}
```

**Transition Properties:**
- **Duration**: 0.3s (300 milliseconds)
- **Timing Function**: `ease-in-out` (slow start, slow end)
- **Properties**: `color` and `transform`

## Areas for Improvement

1. **Extract to external stylesheet** for better caching and reusability
2. **Reduce excessive padding** on `.whatsapp-box` for better responsiveness
3. **Use CSS custom properties** (variables) for color consistency
4. **Add focus states** for keyboard navigation accessibility
5. **Optimize specificity** - avoid overly specific selectors
6. **Consider dark mode** using CSS custom properties and media queries
